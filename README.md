# han
----

## Features

* Minimalist prompt
* Error indicator on return code != 0
* Git support

## Compact vi mode support
* NOTE: these are not the keybindings.
* `>` -- INSERT --
* `<` -- NORMAL -- 
* `v` -- VISUAL -- 
* `_` -- REPLACE -- 

## Configuration
* Set the following variable in your `~/.config/fish/config.fish` in order to use han.
```fish
fish_vi_key_bindings
```
